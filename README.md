# javascript-web-gihub-repos

## Description
WARNING this loads very slowly because
it must fetch alot of data.

Lists all my repos into a searchable datatable
also list all major languages that I use.

## Tech stack
- jquery
- bootstrap
- datatablesJS

## Docker stack
- httpd:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
